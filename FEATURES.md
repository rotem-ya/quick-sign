## בתור — מבוקש ולא ממומש עדיין
- [ ] **התחברות לחשבון Google Drive / OneDrive / אחרים** — אינטגרציית API מלאה
  (רשימת קבצים, פתיחה ושמירה ישירות מהענן בתוך האפליקציה). דורש מבעל המוצר:
  OAuth Client ID ב-Google Cloud Console + רישום אפליקציה ב-Azure (OneDrive).
  הקוד יתבסס על `google_sign_in` + Drive API ו-MSAL. כיום קיים תחליף עובד:
  "שמירה אל…" דרך דיאלוג המערכת.
- [ ] **שמירה ישירה לתיקיית ברירת מחדל** — בחירת תיקייה פעם אחת (SAF
  ACTION_OPEN_DOCUMENT_TREE עם הרשאה מתמשכת), ואז כפתור "שמור" ששומר לשם
  בלחיצה אחת בלי דיאלוג — כולל תיקיית Drive/OneDrive משותפת של הצוות.
- [ ] **צילום חותמת → חותמת דיגיטלית ערוכה** — זיהוי הטקסט בחותמת המצולמת
  (OCR מקומי, למשל ML Kit Text Recognition) ושחזור שלה במעצב הדיגיטלי עם
  אותו מספר שורות, אותו סדר ואותו סגנון — כך שאפשר לתקן שגיאה או לעדכן
  טלפון בלי לצלם מחדש. (הפונט יהיה הקרוב ביותר הזמין — Heebo — לא זהה
  אחד-לאחד לפונט המקורי של החותמת הפיזית.)
