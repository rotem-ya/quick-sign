# quick-sign

אפליקציית **Flutter** לנייד לחתימה על מסמכים — פשטות מוחלטת.

## מה האפליקציה עושה

מקבלים מסמך ל-**quick-sign** דרך שיתוף (Share) של המערכת — קובץ PDF או תמונה —
מקישים במקום שבו רוצים לחתום, נפתח חלון חתימה, וחותמים ביד או באמצעות חותמת שמורה.
אפשר להוסיף הערה, ואז לשתף את המסמך החתום בחזרה.

הכול קורה **מקומית במכשיר** — אין שרת, אין העלאת קבצים, אין צורך בהרשמה או חשבון.

### זרימת עבודה

1. **קבלה** — מקבלים PDF או תמונה דרך תפריט השיתוף של המערכת.
2. **בחירת מיקום** — מקישים על המקום במסמך שבו רוצים לחתום.
3. **חתימה** — נפתח חלון חתימה; חותמים ביד או בוחרים חותמת שמורה.
4. **הערה** — אופציונלי: מוסיפים הערה קצרה.
5. **שיתוף** — משתפים את המסמך החתום בחזרה החוצה.

### היקף ה-MVP

- תמיכה ב-**PDF ובתמונות בלבד**.
- הכול מקומי, ללא שרת, ללא הרשמה.
